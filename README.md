# EcoLearn - Plataforma de Aprendizaje Sostenible

## Descripción
Plataforma educativa con cursos, webinars y tienda de productos sostenibles.

## Características
- Sistema de autenticación de usuarios
- Carrito de compras unificado
- Gestión de productos y cursos
- Sistema de webinars en vivo
- Panel de administración

## Instalación
```bash
pip install -r requirements.txt
python app.py
